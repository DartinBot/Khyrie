# FitFriendsClub Website

🏆 **Premium fitness community website for fitfriendsclub.com**

## 🚀 Quick Start

1. **Start the local server:**
   ```bash
   python3 start_server.py
   ```

2. **Open your browser to:**
   ```
   http://localhost:8000
   ```
